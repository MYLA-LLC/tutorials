title: Dungeons
---

The Dungeons feature in dIVa provides a comprehensive Tier List of Nightmare Dungeons, helping you optimize your gameplay for experience and glyph upgrading.

<span className="center">

![Dungeons Tier List](/images/dIVa/dungeons.png)

</span>

### Functionality

- **Search:** Quickly find specific dungeons using the search bar.
- **Tier Type Filter:** Sort dungeons by their assigned tier type (e.g. Experience).
- **Season Filter:** View dungeons specific to the current season (e.g., 8 - Belial's Return).
- **Dungeon Icons:** Visual representations of each dungeon for easy identification.
- **Dungeon Names:** Clear labeling of each dungeon within its respective tier.

### Tier Types

Dungeons are categorized into the following tier types:

- **Experience:** Sort Dungeons by ones that offer the most experience.

### Using the Dungeons Feature

1. Open the Dungeons window in dIVa by clicking the Skull icon in the sidebar.
2. Use the search bar to find specific dungeons or browse the tier list.
3. Click on the Tier dropdown to filter dungeons by their assigned tier type.
4. Use the Season dropdown to view dungeons relevant to the current season.
