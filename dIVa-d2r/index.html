Desktop

Screenshot of the Desktop window.

Functionality

  • Experience Meter: This meter can run either in-game or as part of the desktop window.
  • Speed Leveling Tracker: Monitors the speed at which your character levels up. Displays time for every Milestone!
  • Settings:
    • Icon: Displays a red cog when no character is detected, or shows the current character's class icon when a character is recognized.
    • Hover: Shows the current character's name and level when hovered over.

Characters

Functionality

  • Character List: View and manage your characters. Each character entry displays the name, current status (e.g., "Main," "Please select Favorite"), and offers options to edit, delete, or set as favorite.
  • Active Character: The top section shows the currently active character, displaying their name and class icon.
    If no character is detected, a red cog icon is shown with the message "No Character Detected."
  • Alias Information: When a character has multiple aliases, the alias information along with the last known level and available actions (edit, delete, mark as favorite) is displayed under the character’s name.
  • Character Creation: Create new characters directly within the Characters feature using the modal window. You can select the class, gender, character name, starting level, and mark the character as "Offline" if needed.
  • Offline Mode Integration: Characters added while offline are automatically marked as "offline," and you can manage offline character progression through the new "Offline Mode" feature.
  • Character Management: Easily update aliases or delete characters, ensuring your character list remains accurate and up to date.

Meters

Screenshots of the Experience Meter

Functionality

  • Total Resource: Displays your current experience collected in the session.
  • Resource Per Second: Displays current experience per second.
  • Total Time: Shows the total time the meter has been running.
  • Session High: Displays the highest amount of experience gained in a session.
  • Lifetime Average: Displays the lifetime average experience per second.
  • In-Game Overlay: Shows a limited subset of this information when used as an in-game overlay.

Speed Leveling Tracker

Screenshot of the Speed Leveling Tracker window.

Functionality

  • Milestones: Displays milestones along your leveling journey.
    • Total Time: The time taken within the milestone; accumulative of all the levels.
    • Level Breakdown: Shows the elapsed time for each level within the milestone.
  • Elapsed Time: Displays the elapsed time for each level individually.
  • Progress Tracking: Tracks the time taken to reach each milestone and displays it in an expandable list.
  • Control Button: Includes a play/pause button to start or stop the timer.

Settings

Screenshot of the Settings window.

Functionality

  • Configuration: Adjust application settings.
    • Leveling Tracker Auto-start: Automatically starts the Speed Leveling Tracker when a character logs in.
    • Experience Overlay: Enables the Experience Meter as an in-game overlay.
  • Reset Data: Clear accumulated data in the application.
    • Experience: Resets all experience-related data.
    • Delete All: Deletes all data associated with the application, including settings, resource data per character, leveling data per character, and all characters.
  • Manage Subscription: Update subscription details.
    • Edit: Modify subscription settings and details.
