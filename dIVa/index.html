v0.66.6 - September 2024

Discover the exciting new features and improvements included in dIVa v0.66.6!

New Features

  • Multi-Game Support: dIVa is expanding beyond Diablo IV! Now supporting Diablo II: Resurrected.
  • New "Characters" Feature: A dedicated interface for managing character and their aliases.
  • Leveling Overlay: An in-game overlay to track and manage leveling milestones.
  • Offline Mode: Manage Characters and the Speed Leveling Tracker even when Overwolf game events are unavailable.
  • Multi-Language Support: dIVa now supports multiple languages, allowing you to enjoy the full experience in your preferred language!

Improvements

  • Menu Cleanup and Consolidation: Streamlined the application menu to support the new "Characters" feature.
